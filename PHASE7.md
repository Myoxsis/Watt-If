# Phase 7 — Stretch Features

Implemented in the Node/Vite app.

## Added

- Compare your simulated grid to France, Germany, and Sweden benchmarks
- Real-world inspired country presets
- Export the current scenario summary as an SVG image
- Phase 7 styling for presets, benchmark cards, and export controls

## Files

- `src/data/countryPresets.ts`
- `src/components/Phase7Stretch.tsx`
- `src/phase7.css`
- `src/App.tsx`
- `src/main.tsx`

## Notes

The country presets are simplified, real-world inspired profiles intended for hackathon storytelling, not official grid datasets.

```bash
npm install
npm run dev
```
