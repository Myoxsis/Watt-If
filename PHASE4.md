# Phase 4 — Game Loop

Implemented in the Node/Vite app.

## Added

- Yearly turns from 2026 to 2040
- Starting budget and annual budget refresh
- Investment cards for solar, wind, batteries, nuclear, hydro, and efficiency
- Automatic crisis event at the end of each year
- Annual timeline showing score, risk, event, budget, and investments
- Final 2040 report card
- Replay flow

## Run

```bash
npm install
npm run dev
```

## Build check

```bash
npm run build
```
